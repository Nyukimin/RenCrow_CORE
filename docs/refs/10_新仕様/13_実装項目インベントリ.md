# 実装項目インベントリ

## 目的

この文書は、RenCrow 現行実装の主要項目を `docs/10_新仕様/` だけで追えるようにするための一覧である。

旧 docs は履歴参照であり、現行実装判断の入口はこの新仕様セットとする。

## 状態の定義

| 状態 | 意味 |
| --- | --- |
| 実装済み | production code と主要配線が存在し、通常利用できる |
| 部分実装 | 主要機能はあるが、未接続、限定運用、残作業、未整理の境界がある |
| 未実装 | 現行 production code としては存在しない |
| 旧仕様・移行対象外 | 過去構想または別体系であり、新規 repo の現行仕様入口には持ち込まない |
