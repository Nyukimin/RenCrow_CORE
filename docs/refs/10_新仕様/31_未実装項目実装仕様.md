- 内容: 旧 OpenClaw 由来 Docker sandbox は移行対象外候補。

#### 対立点

#### 優先判断

`29_Sandbox_Promotion_Gate仕様.md` を優先する。Docker は必須ではなく、MVP は worktree / sandbox root / Path Guard とする。

#### 実装仕様での扱い

次に Sandbox Promotion Gate を実装する。理由は、以後の AI Native / Workstream / Skill 実装で write や artifact 生成が増えるため、正式環境へ出す前の gate を先に置く必要があるためである。
