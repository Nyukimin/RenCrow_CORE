## 概要

RenCrowの結合は、`cmd/picoclaw`をcomposition rootとしてHTTP adapter、application/domain、infrastructure/store/providerへ広がる。Viewer surfaceが最大の結合点で、会話・音声・記憶・job・evidence・運用タブが`/viewer/*`とSSE eventに集まる。

## アーキテクチャ概要

### Clean Architecture データフロー

```text
HTTP/channel/viewer input
  -> cmd/picoclaw route registration
  -> internal/adapter/*
  -> internal/application/* use case / orchestrator
  -> internal/domain/* entity/value/contract
  -> internal/infrastructure/* provider/store/tool runner
  -> event/evidence/memory/job output
  -> viewer SSE / HTTP response / persisted store
```

### runtime composition

```text
config + workspace
  -> cmd/picoclaw/runtime_*.go
  -> Dependencies
  -> routes.go
  -> handler/service/store/provider
```

## 主要チェーン

### Viewer会話送信

```text
viewer.js / /viewer/send
  -> internal/adapter/viewer.HandleSendWithAttachments
  -> MessageHandler implementation in cmd dependencies
  -> internal/application/orchestrator.MessageOrchestrator
  -> routing/domain agent/provider/tool/memory
  -> EventHub / MonitorStore / response JSON
```

### Viewer live update

```text
orchestrator.NewEvent(...)
  -> viewer.EventHub.OnEvent
  -> MonitorStore reducers
  -> /viewer/events SSE
  -> Viewer JS tabs
```

### STT/Voice Chat

```text
browser mic / viewer STT endpoints
  -> internal/adapter/viewer STT capture/admin handlers
  -> modules/stt session rules and runtime observer
  -> orchestrator.ProcessVoiceDirect
  -> TTS/audio route/EventHub
```

### Knowledge/Memory Recall

```text
docs/wiki + conversation L1 stores + user memory
  -> internal/infrastructure/persistence/conversation
  -> internal/adapter/viewer memory handlers
  -> /viewer/memory/recall-pack
  -> orchestrator prompt/context injection
```

## Middleware / Reducer / Observer相当の実行順

このrepoはDuxではなくClean Architecture + event reducer構成であるため、以下を相当概念として扱う。

| 種別 | 実装 | 処理内容 |
|---|---|---|
| Middleware相当 | `internal/infrastructure/llm/middleware/*` | LLM requestへdatetime/context budget/raw log等を適用。 |
| Middleware相当 | `internal/infrastructure/tools/*` | Tool executionへsandbox/context budget/harnessを適用。 |
| Reducer相当 | `internal/adapter/viewer/monitor_reducers.go` | orchestrator eventからViewer monitor stateを更新。 |
| Observer相当 | `internal/adapter/viewer.EventHub` | orchestrator eventをSSE/history/client countへ反映。 |
| Observer相当 | background jobs / notification listener | job失敗やscheduled taskをeventとして通知。 |
