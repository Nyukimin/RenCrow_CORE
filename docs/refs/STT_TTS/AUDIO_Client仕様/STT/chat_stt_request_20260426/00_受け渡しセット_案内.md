このフォルダは、Chat 実装担当へ RenCrow_STT の確定仕様を渡すための最小セットです。

## 同梱ファイル

1. `01_依頼文_Chat本体向け.md`  
   Chat 側へそのまま送れる依頼文
2. `02_STT_API.md`  
   WebSocket/HTTP 契約（実装合わせ）
3. `03_STT仕様.md`  
   運用前提・責務分界・エラー取り扱い
4. `04_CHAT統合契約.md`  
   Chat 側の実装責務と受け入れ条件
5. `05_起動手順_README.md`  
   接続確認の手順と確認コマンド
6. `06_仕様_Chat向け.md`  
   Chat 向けに抜粋した確定仕様
7. `07_実装仕様_Chat向け.md`  
   Chat が把握すべき実装仕様
8. `README.md`  
   Chat 連携用の入口 README

## この受け渡しで確定したポイント

- 主経路は `ws://<host>:8090/stt`（後方互換: `/ws`, `/stt-ws`）
- Chat は `final` を受け取って LLM/TTS へ接続する（`reply_*` は存在しない）
- LAN/リモート運用は WSS 推奨（ローカル検証のみ WS 許容）
