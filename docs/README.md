## 文書の位置づけ

- `main` のこの10ファイルだけを、Public向けか内部向けかを問わない現行の製品仕様正本とします。
- `AGENTS.md`、`CLAUDE.md`、`rules/` は作業者向けの実行制約です。製品仕様を再定義せず、この正本を参照します。
- 実装、production wiring、test、config は現在状態を確認する証拠です。正本と差異が見つかった場合は事実を照合し、採用する契約をこの正本へ反映してから実装を合わせます。
- 実装済み、未実装、deployment 依存、再採用判断が必要な旧構想を混同しません。
- 整理前の全資料と分類一覧を保存した `archive/docs-classified-20260715` と、LLM向けの選別済みKnowledgeである `knowledge/rencrow-core` は履歴・参照資料であり、現行判断の正本にしません。
- 現行正本に必要な情報が不足している場合は、該当する `01` から `09` の文書を更新します。別の正本ディレクトリ、版付き正本、補助正本を追加しません。
