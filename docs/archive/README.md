## 役割

このフォルダは、通常参照から外した旧資料を保存する場所です。削除ではありません。

`docs/` を通常入口、`docs/refs/` を必要時の参照置き場とし、どこからも参照されていない旧資料はここへ移動します。

## ルール

- archive 内の資料を正本として扱わない。
- 復帰させる場合は、理由、関係する正本、一時参照か恒久追加か、いつ外すかを DOC チケットへ記録する。
- 旧資料を復帰させる前に、`docs/99_整理/最小化判定.md` と `docs/復帰候補一覧.md` を確認する。
