## 背景

`01_理解/02_キャラクター・エージェント仕様.md` では Kin を Coder4 / `CODE4` として説明している。一方、`02_正本仕様/01_仕様.md` のルーティングカテゴリは `CODE / CODE1 / CODE2 / CODE3` を中心に書かれている。
