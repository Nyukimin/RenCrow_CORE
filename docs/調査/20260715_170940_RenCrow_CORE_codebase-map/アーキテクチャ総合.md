# RenCrow_CORE アーキテクチャ総合

## 概要

RenCrow_COREは、複数agentと外部moduleを接続するruntime/composition systemである。現在はmodules contract、feature facade、legacy-bodyが共存する移行期で、非破壊と観測可能性を優先している。
