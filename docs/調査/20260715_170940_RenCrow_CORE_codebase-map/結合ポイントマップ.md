# RenCrow_CORE 結合ポイントマップ

## 概要

主要結合点はcomposition root、feature registrar、MessageOrchestrator、EventHub、ToolRunner、ConversationManagerである。依存はmodules/domain contractへ向け、external I/Oはinfrastructureへ閉じる。

## Middleware チェーン（実行順）

| 順 | 境界 | 処理 |
| ---: | --- | --- |
| 1 | HTTP/Tailscale guard | 到達範囲 |
| 2 | handler validation | payload/size/path |
| 3 | route/policy | Chat/Worker/Coder権限 |
| 4 | Tool Harness/sandbox | 副作用可否 |
| 5 | provider/store | external I/O |

## Reducer チェーン（実行順）

専用Reducer chainは全体共通ではない。EventHub→MonitorStore reducer、conversation→Memory lifecycle、TTS→playback stateが個別state ownerを持つ。

## Observer チェーン

| observer | 入力 | 出力 |
| --- | --- | --- |
| EventHub | OrchestratorEvent | SSE sequence/history |
| MonitorStore | OrchestratorEvent | Viewer projection |
| RecallTraceStore | RecallPack decisions | trace persistence |
| TTS playback observer | audio events/ACK | playback state |
| Heartbeat listener | scheduled result | logs/events |
