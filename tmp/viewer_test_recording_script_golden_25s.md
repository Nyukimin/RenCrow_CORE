## 録音の流れ

1. Viewer を **ハードリロード**
2. **Ops** → **Runtime / LLM / Audio** → **STT テスト録音**
3. **録音開始**（TTS / IdleChat が止まることを確認）
4. 上の読み上げ文を **1 回通し** で自然な速度で読む
5. 読み終わったら **1 秒ほど黙る**
6. **録音停止・保存**
